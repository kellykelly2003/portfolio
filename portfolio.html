What do I do?

I am a full-stack developer

My front-end design is used to convey customers' branding and needs to make a business easily recognizable. It is an ever changing field that requires continuous education that I enthusiastically embrace. Many skills are needed to be competitive and HTML, CSS, and Javascript are essential building blocks that I use as base front-end design tools.

My back-end programming implements technical skills that allow front-end designs to work effortlessly. Developing applications and/or websites require a rigorous routine that involves creating, maintaining, debugging, and testing for functionality. PHP, C#, SQL and Python are some of the back-end programming languages I use to add utility to everything created in the front-end.
