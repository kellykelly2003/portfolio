Responsive Design

The Details

Built using HTML & CSS.

Responsive design is important to ensure a website is built, with the conscious decision, to function properly across multiple devices and screen sizes. These are simple pages created using different responsive techniques (including CSS Grids, FlexBox, and smooth scrolling). These allow the user's screen to change sizes and the items to re-arrange themselves on the page, to hover over items and that item to grow a little larger so the user is sure what they are doing, and the ability scroll up/down a page without being distracted by jerky screen movements.
