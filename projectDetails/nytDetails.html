New York Times API

The Details

I used the New York Times Application Program Interface(API) to build an app that allows the user to choose any topic from a dropdown menu and display all related articles. My application is styled after an old fashioned newspaper and given a vintage color palette to maintain that affect.

While this may look like an old fashioned newspaper, but was created using cutting edge software, programming languages, and skills. I used JavaScript, HTML, CSS, Bootstrap, design elements, and the API from New York Times. Special care was taken to make the app responsive at all sizes (mobile, tablet, and desktop) and that the user's experience was a priority.
